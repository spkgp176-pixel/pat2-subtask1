

## Overview
This project demonstrates the use of Morse code translation using C++. 
The program converts English text into Morse code symbols.

## Historical Background
Morse code was developed in the 1830s and 1840s by Samuel Morse and Alfred Vail. 
It was mainly used for telecommunication through telegraphs.

## System Working
The program:
1. Accepts user input
2. Reads each character
3. Converts characters into Morse code
4. Displays the translated message
